# Page de renvoi de MonComptePro

## Accèder à la page

[https://mon-compte-pro.beta.gouv.fr](https://mon-compte-pro.beta.gouv.fr)

## Requirements

- NodeJs v16

## Installation

```shell
git clone git@github.com:betagouv/mon-compte-pro-landing-page.git
cd mon-compte-pro-landing-page
npm i
npm run build
npm start
```
