# Page de renvoi de MonComptePro

## Accèder à la page

[https://moncomptepro.beta.gouv.fr](https://moncomptepro.beta.gouv.fr)

## Requirements

- NodeJs v16

## Installation

```shell
git clone git@github.com:betagouv/moncomptepro-landing-page.git
cd moncomptepro-landing-page
npm i
npm run build
npm start
```
